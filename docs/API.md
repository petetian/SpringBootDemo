# API Reference

Base URL: `http://localhost:8080`

## Endpoints

### `GET /`

Returns service liveness text.

Response body:

```text
Spring boot is running!
```

### `GET /greeting`

Returns a sample greeting text.

Response body:

```text
Hello from Microsoft
```

### `GET /customers`

Returns all customers from the database.

Response body:

```json
[
  {
    "id": 1,
    "firstName": "John",
    "lastName": "Doe",
    "accountNumber": "A1238374",
    "createdAt": "2026-06-03T12:00:00",
    "updatedAt": "2026-06-03T12:00:00"
  }
]
```

Error behavior:

- Returns HTTP `500` if the read operation fails.

### `POST /create`

Creates a customer.

Request body:

```json
{
  "firstName": "Alice",
  "lastName": "Walker",
  "accountNumber": "A100200"
}
```

Response body:

- Created `Customer` entity JSON (same shape as `GET /customers` item).

## Operational Endpoints

### `GET /actuator/health`

Spring Boot Actuator health status.

### `GET /api-docs`

OpenAPI JSON generated by springdoc.

### `GET /swagger-ui/index.html`

Swagger UI web interface.

### `GET /h2-console`

H2 console UI (enabled for local development).
